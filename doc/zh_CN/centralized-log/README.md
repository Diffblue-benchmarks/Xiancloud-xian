### graylog
graylog是一个优秀的开源日志管理系统
- graylog提供了优雅的图形界面来展示日志
- graylog提供了ldap、active directory接入认证
- graylog提供了基于RBAC（role based access control）权限管理能力
有些graylog追随者这样评价它：
> graylog简直就是开源版本的[splunk](https://www.splunk.com/)
